Quest Keyer

Program Info

This program is a chroma keying application. It supports keying red, green, or blue backgrounds and uses Petro Vlahos' Ultimatte keying technique. The program utilizes an Electron frontend, a C++ Drogon API, and the C++ OpenCV SDK for the keying, image manipulation, and image IO functionality.

Version

v0.0.1

Developer Info

This program was developed by Noah Turnquist in 2024 as a portfolio project.
